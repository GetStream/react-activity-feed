IconBadge

IconBadge is technically a wrapper that displays a badge with or without numbers on it's children.

Properties

hidden
boolean | undefined
onClick
((event: MouseEvent<HTMLDivElement, MouseEvent>) => void) | undefined
showNumber
boolean | undefined
unseen
number | undefined

Basic usage

<IconBadge onClick={() => console.log('hello world.')} />

With Unseen

<IconBadge unseen={3} />

With Numbers

3

<IconBadge unseen={3} showNumber />

Custom Icon

3

<IconBadge showNumber unseen={3}>
  <div
    style={{
      width: 20,
      height: 20,
      borderRadius: '50%',
      backgroundColor: 'hotpink',
    }}
  />
</IconBadge>

Hidden

<IconBadge showNumber unseen={3} hidden />
